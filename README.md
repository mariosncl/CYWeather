# CYWeather 🌤️

A simple JavaFX weather application for Cyprus.

## Features

- JavaFX graphical interface
- Live weather data using OpenWeather API
- Search Cyprus cities
- Displays:
  - Temperature
  - Weather condition
  - City


## Technologies

- Java 21
- JavaFX
- Maven
- OpenWeather API


## Setup

1. Create an account at:

https://openweathermap.org/


2. Generate an API key.


3. Replace:

YOUR_OPENWEATHER_API_KEY

inside:

WeatherService.java


4. Run:

mvn clean javafx:run

- Limassol
